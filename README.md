# How to get started

1. Download the [uv project and package manager](https://docs.astral.sh/uv/)
2. Create a python virtual environment using `uv venv` and activate it with `source .venv/bin/activate` 
3. Installed the packages defined in the uv.lock file and pyproject.toml file using `uv sync --locked`

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
